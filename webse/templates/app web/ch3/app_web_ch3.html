Chapter 3. Navbar

This chapter consists on three parts. First, in the layout.html file, we create a basic navigation bar by using HTML code. With the navigation bar, the user can move to the home and the app calculator webpages. Second, in the head of the layout.html file, we introduce some CSS style classes to make our navigation bar more appealing and functional. Finally, to clean the layout.html, and to make our code as efficient and functional as possible, we place all the CSS style classes in a separated file (style_carbon.css) located in a separated folder (static).

Navbar, HTML

In the layout.html file, we introduce a basic navigation bar by using HTML code.

  1. Download the entire folder (Ch3.a. Navbar HTML Initial) with the project (virtual environment + rest of the files) that corresponds to your operative system (Windows or Mac). Be sure that you download the INITIAL folder.
    OneDrive, Ch3.a. Navbar HTML Initial
  2. In PyCharm, open the entire project (File>Open>(Select the folder that you have downloaded in the previous step).
  3. Follow the instructions in the link to video 1. In that video, I explain how to add code step by step until you have a basic navigation bar where you can move to the home and the app calculator web pages.
    Video 1: Navbar HTML

  4. Run the app
    Run the app

    1. In PyCharm, open the terminal and activate the virtual environment.
        In Windows: (venv\Scripts\activate)
        In Mac: (source venv/bin/activate)

    2. By using the terminal in Pycharm, install flask: (pip install flask) (or pip3 install flask). The command pip (or pip3) depends on the Python version installed on your computer. Try both.

    3. By using the terminal in Pycharm, run your App: (python app.py) (or phyton3 app.py). The command python (or python3) depends on the Python version installed on your computer. Try both.

    4. Open the app in your browser: Open the blue link that appears in Pycharm after running your app.py file to open your app in the browser.

  5. The final version of the code can be downloaded from the Github account and from the OneDrive folder. In the OneDrive folder, it is also included the virtual environment. To run the final version, follow the steps described in the previous step.
