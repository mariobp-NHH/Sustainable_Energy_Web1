Instructions to run your first App

Please follow the next steps to run your first App:

  1. Download the entire folder with the project (virtual environment + app.py file) that corresponds to your operative system (Windows or Mac).
    OneDrive, Ch1
  2. In PyCharm, open the entire project (File>Open>(Select the folder that you have downloaded in the previous step).

  3. In PyCharm, open the terminal and activate the virtual environment.
      In Windows: (venv\Scripts\activate)
      In Mac: (source venv/bin/activate)

  4. By using the terminal in Pycharm, install flask: (pip install flask) (or pip3 install flask). The command pip (or pip3) depends on the Python version installed on your computer. Try both.

  5. By using the terminal in Pycharm, run your App: (python app.py) (or phyton3 app.py). The command python (or python3) depends on the Python version installed on your computer. Try both.

  6. Open the app in your browser: Open the blue link that appears in Pycharm after running your app.py file to open your app in the browser.
