App Development Module

This module contains all the chapters that we will use to develop our carbon emissions app calculator.

In this module we develop a complete app by installing Python, Flask, choosing an editor to develop the app, and deploying the app by using Github and Heroku.

We design the esthetics of the webpage by using HTML and CSS, and the economic calculations are conducted by using Python.

The aim of the module is to focus on the economic models used to work out transport carbon emissions, and connect that economic model with the development of an app.
