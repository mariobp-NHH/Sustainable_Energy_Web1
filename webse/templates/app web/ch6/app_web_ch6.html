Chapter 6. Forms

In this chapter, we allow the user to interact with our app by selecting the means of conveyance, the type of fuel in a particular mean of conveyance, and the number of kilometers travelled by using that mean of conveyance.

To develop a Flask form, we proceed in two steps. First, we program the form in two subsequent steps. First, we create that form in a separated form.py file; second, we connect that form with our routes in the app.py file. Second, we design and connect the form with the HTML code in the calculator.html file.

Forms, programming

To program the flask form, we proceed in two steps:

  1. We create a file named forms.py where we import all the tools that we will need to create our form; we create the form as a class; and we define all the elements that we will use in our form.
  2. We connect the form.py with our routes by using the app.py file.
    1. We import the file forms.py that we created in the previous step: from forms import AddRecordForm
    2. We create a secret key: app.config['SECRET_KEY'] = '3434345791628bb0b13ce0c676dfde280ba245'
    3. In the route where we will use the form, we introduce the methods get and post in such a way that the users can interact with our app: @app.route('/app_calculator', methods=['GET', 'POST'])
    4. We pass the information of our form to the app_caculator.html:
