Chapter 2. Templates in the HTML code

This chapter consists on three parts. First, in the routes file, we develop a complete HTML code with the content of each of the sections of our webpage. Second, to keep the route file as simple and organized as possible, we place the HTML code in a separated HTML file that is stored in the template folder. Finally, to avoid repetition, to simplify the HTML code and to increase the efficiency of our code, we introduce templates.
