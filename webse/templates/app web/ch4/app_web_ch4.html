Chapter 4. Header

This chapter consists on three parts. First, in the home.html file, we define the HTML code for the header of our home page. Second, in the layout.html file, we define the CSS classes that determine final view of our home page. Third, we complement our own CSS classes with the classes defined in Bootstrap, and we learn how to combine our own HTML code, with our own CSS classes and the CSS classes defined in Bootstrap.

We conclude the chapter completing the home.html introducing the HTML code that define the developer and the footer sections. We also define our own CSS classes and the Bootstrap CSS classes to make those sections more appealing and functional.

Header, HTML

In the home.html file, we design the structure of our header by using HTML code.

  1. From OneDrive, download the entire folder (Ch4.a. Header HTML Initial). That folder contains all the files and sub-folders of our project. That folder contains the recipe to make the cake. Be sure that you download the INITIAL folder.
    OneDrive, Ch4.a. Header HTML Initial
  2. Copy the folder that you have downloaded and place it in the folder in your computer where you have the rest of information of the course.
  3. The folder that you have downloaded contain the recipe of the cake, but you need to add to that folder the tools to make that cake. Therefore, you need to add the VIRTUAL ENVIRONMENT that correspond to your operative system.

    You have two options. The easier way is to copy the virtual environment from one of the folders in your own computer from one of the previous chapters and paste it in the folder that you have created that contain the recipe of the cake. The second option, is to download it and copy it from the OneDrive link:
    OneDrive. Virtual Environment
  4. In PyCharm, open the folder that contain the recipe and the tools (files and sub-folders of the project + virtual environment) (File>Open>(Select the folder in which you have the files and sub-folders of the project + the vitual environment)).
  5. Follow the instructions in the link to video 1. In that video, I explain how to add code step by step until you define a basic header for your project
    Video 1: Header HTML

  6. Run the app
    Run the app

    1. In Pycharm, open the terminal and be sure that you are in the folder of your project.

      If you are not in the folder of your project, close the terminal, and open it again. By doing that, PyCharm will open the terminal in the folder where you have your project.

    2. In the terminal, activate the virtual environment.
        In Windows: (venv\Scripts\activate)
        In Mac: (source venv/bin/activate)

    3. By using the terminal in Pycharm, install flask: (pip install flask) (or pip3 install flask). The command pip (or pip3) depends on the Python version installed on your computer. Try both.

    4. By using the terminal in Pycharm, run your App: (python app.py) (or phyton3 app.py). The command python (or python3) depends on the Python version installed on your computer. Try both.

    5. Open the app in your browser: Open the blue link that appears in Pycharm after running your app.py file to open your app in the browser.

  7. The final version of the code can be downloaded from the Github account and from the OneDrive folder. Do not forget to add the virtual environment to the folder where you have all the files and sub-folders of your project. To run the final version, follow the steps described in the previous step.
